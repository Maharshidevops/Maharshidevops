# 🔥 IMPORTANT (Pacman Setup)

👉 The animation works **only if GitHub Action is enabled**

### Step to enable:

1. Go to your repo: `Maharshidevops`
2. Create folder:

```
.github/workflows
```

3. Create file: `snake.yml`

Paste this:

```yaml
name: Generate Snake

on:
  schedule:
    - cron: "0 */12 * * *"
  workflow_dispatch:

jobs:
  build:
    runs-on: ubuntu-latest
    steps:
      - uses: Platane/snk@v3
        with:
          github_user_name: Maharshidevops
          outputs: |
            dist/github-contribution-grid-snake-dark.svg

      - uses: crazy-max/ghaction-github-pages@v3
        with:
          target_branch: output
          build_dir: dist
        env:
          GITHUB_TOKEN: ${{ secrets.GITHUB_TOKEN }}
```
